# Reference image inputs

Place the product images for this example in this directory. The bundled YAML files expect these names:

```text
chiller-front.jpg
chiller-angle.jpg
product-size-reference.jpg
scene-reference-01.jpg
scene-reference-02.jpg
```

Optional style and detail references can also be added:

```text
plunge-style-reference.png
product-details-reference.jpg
```

Use images from only one tub SKU in each configuration. Keep clean white-background front and three-quarter chiller photos as the primary identity references. Do not publish supplier images unless you own them or have permission to use them.
